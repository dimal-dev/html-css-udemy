<Question 2>
    <Description>
        You want a paragraph to be red. Which of the following CSS rules will achieve that?
    </Description>
    <Answer 1>
        <Is Correct>No</Is Correct>
        <Content>p { background-color: red; }</Content>
        <Explanation>Why? This changes the background color, not the text color.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>No</Is Correct>
        <Content>.text-red { text-color: red; }</Content>
        <Explanation>Why? text-color is not a valid CSS property. The correct one is just color.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>Yes</Is Correct>
        <Content>.text-red { color: red; }</Content>
        <Explanation>Why? This is the correct syntax for changing text color using a class selector.</Explanation>
