<Question 2>
    <Description>
        You want to make only the second paragraph on a page red. What selector should you use?
    </Description>
    <Answer 1>
        <Is Correct>Yes</Is Correct>
        <Content>.text-red</Content>
        <Explanation>Why? A class selector like .text-red lets you target just that specific paragraph.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>No</Is Correct>
        <Content>p</Content>
        <Explanation>Why? This would target all paragraph elements, not just the second one.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>No</Is Correct>
        <Content>#text-red</Content>
        <Explanation>Why? This would select an element with ID "text-red", but the class uses a dot, not a hash.</Explanation>
    </Answer 3>

<Question 3>
    <Description>

    </Description>
    <Answer 1>
        <Is Correct>No</Is Correct>
        <Content>text-transform</Content>
        <Explanation>Why? text-transform changes the case of text, not its size.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>No</Is Correct>
        <Content>line-height</Content>
        <Explanation>Why? line-height adjusts spacing between lines, not the font size.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>Yes</Is Correct>
        <Content>font-size</Content>
        <Explanation>Why? font-size is exactly what you use to change the size of text.</Explanation>

<Question 5>
    <Description>
        Which of the following is the correct way to write a class selector in CSS?
    </Description>
    <Answer 1>
        <Is Correct>No</Is Correct>
        <Content>class="text-big"</Content>
        <Explanation>Why? That’s HTML syntax for assigning a class, not a selector in CSS.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>No</Is Correct>
        <Content>p.text-big</Content>
        <Explanation>Why? This is a valid combined selector, but it's not the simplest standalone class selector.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>Yes</Is Correct>
        <Content>.text-big</Content>
        <Explanation>Why? .text-big correctly selects any element with the class "text-big" in CSS.</Explanation>
