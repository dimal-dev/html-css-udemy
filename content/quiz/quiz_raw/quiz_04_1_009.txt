<Question 2>
    <Description>
        You want all images on your webpage to be displayed at 250 pixels wide. What kind of CSS selector should you use?
    </Description>
    <Answer 1>
        <Is Correct>Yes</Is Correct>
        <Content>tag selector</Content>
        <Explanation>Why? A tag selector like img applies styles to all HTML elements of that type on the page.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>No</Is Correct>
        <Content>id selector</Content>
        <Explanation>Why? An ID selector targets only one specific element with a unique ID.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>No</Is Correct>
        <Content>class selector</Content>
        <Explanation>Why? A class selector applies styles to elements with a specific class name, not all elements of that type.</Explanation>
    </Answer 3>

</Question 2>

<Question 3>
    <Description>
        Real-world scenario: You added a second image and noticed it’s automatically resized just like the first one. Why is that happening?
    </Description>
    <Answer 1>
        <Is Correct>No</Is Correct>
        <Content>They have different URLs, but the browser resizes them the same by default.</Content>
        <Explanation>Why? The browser doesn’t automatically resize images based on similarity. CSS selectors do that.</Explanation>
    </Answer 1>
    <Answer 2>
        <Is Correct>Yes</Is Correct>
        <Content>The CSS tag selector img applies the same width property to both images.</Content>
        <Explanation>Why? The tag selector targets all img elements, so any new image inherits the same CSS rule.</Explanation>
    </Answer 2>
    <Answer 3>
        <Is Correct>No</Is Correct>
        <Content>The HTML automatically adjusts size when there are multiple images.</Content>
        <Explanation>Why? HTML doesn’t adjust size on its own; only CSS can style multiple elements this way.</Explanation>
    </Answer 3>
